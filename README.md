Crypto Currency Tracker is a React.js web application built with Vite that provides real-time cryptocurrency market data. The website displays the top 100 cryptocurrencies with pagination, search, and sorting capabilities, allowing users to filter coins by rank, market volume, and price. Each cryptocurrency has a dedicated details page where users can view comprehensive market information along with a line chart visualizing the last 7 days of price history with corresponding dates. The application focuses on performance, usability, and clean UI to deliver an efficient and informative crypto-tracking experience.
